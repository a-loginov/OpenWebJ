# OpenWebJ

OpenWebJ — это современное веб-приложение для создания и организации заметок, вдохновленное Notion. Приложение предоставляет интуитивный интерфейс для создания страниц с различными типами контента.

## Возможности

- **Создание страниц** с настраиваемыми заголовками, иконками и обложками
- **Блочная система контента** с поддержкой различных типов блоков:
  - Текстовые блоки
  - Заголовки (H1, H2, H3)
  - Маркированные списки
  - Чекбоксы для задач
  - Блоки кода с подсветкой синтаксиса
  - Цитаты
- **Организация страниц** по категориям
- **Различные виды отображения**:
  - Dashboard — карточки всех страниц
  - Calendar — календарный вид
- **Темная и светлая темы**
- **Горячие клавиши** для быстрой работы
- **Сворачиваемая боковая панель** для комфортной работы

## Технологии

- HTML
- CSS (с поддержкой CSS-переменных для тем)
- Vanilla JavaScript (без внешних зависимостей)
- LocalStorage для хранения данных

## Установка и запуск

### Вариант 1: Открытие напрямую

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/OpenWebJ.git
cd OpenWebJ
```

2. Откройте файл `templates/index.html` в браузере

### Вариант 2: Запуск через локальный сервер

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/OpenWebJ.git
cd OpenWebJ
```

2. Запустите локальный сервер (Python):
```bash
# Python 3
python -m http.server 8000
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:8000/templates/index.html
```

### Вариант 3: Flask-приложение

Если вы хотите использовать Flask:

1. Создайте виртуальное окружение и активируйте его:
```bash
python -m venv .venv
source .venv/bin/activate  # для Linux/Mac
# или
.venv\Scripts\activate  # для Windows
```

2. Установите Flask:
```bash
pip install flask
```

3. Создайте файл `app.py`:
```python
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)
```


5. Откройте браузер и перейдите по адресу `http://localhost:5000`

## Использование

### Создание новой страницы

1. Нажмите кнопку "Новая страница" в боковой панели или используйте `Ctrl+N`
2. Введите заголовок страницы
3. Начните добавлять блоки контента

### Добавление блоков

1. Нажмите кнопку "Добавить блок" или `+` под последним блоком
2. Выберите тип блока из меню:
   - **Текст** — обычный текстовый блок
   - **Заголовок 1-3** — заголовки разного уровня
   - **Список** — маркированный список
   - **Чекбокс** — список задач
   - **Код** — блок кода с подсветкой
   - **Цитата** — выделенная цитата

### Настройка страницы

1. Нажмите кнопку "Настройки" в заголовке страницы
2. Настройте:
   - **Иконку** — эмодзи для страницы
   - **Цвет** — цветовая метка
   - **Категорию** — группа страницы
   - **Обложку** — градиентная обложка

### Переключение видов

Используйте кнопки в боковой панели:
- **Dashboard** — просмотр всех страниц в виде карточек
- **Calendar** — календарный вид страниц

## Хранение данных

Все данные сохраняются локально в браузере с использованием LocalStorage. Это означает:
- Данные доступны только в текущем браузере
- Данные сохраняются автоматически
- Для синхронизации между устройствами требуется дополнительная интеграция

## Разработка

### Архитектура

Приложение построено на модульной архитектуре:

- **index.html** — структура приложения
- **style.css** — стили с использованием CSS-переменных для тем
- **script.js** — логика приложения, управление состоянием

### Основные компоненты

1. **PageManager** — управление страницами
2. **BlockManager** — управление блоками контента
3. **ThemeManager** — управление темами
4. **StorageManager** — работа с LocalStorage
